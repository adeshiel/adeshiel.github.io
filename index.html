B-Essay

Ever needed to meet a word count, but never knew what else to write?
Ever too tired to try and make it up on your own?
Never heard of the "white-space" trick, like me?

That's where B-Essay steps in!

It simply extends the vocabulary in a pre-existing essay to meet the necessary word count in a slightly smart manner.


This was the first web-app I've ever made, and was simply inspired by my writing class freshman year. I want to add machine learning capabilities to it later down the line to make it much smarter.

Technologies: Python 2.7, Word API by Mashape, HTML, CSS
